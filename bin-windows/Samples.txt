
>cprime hello.c -o hello2.c

This will generate the hello2.c from hello.c



>cprime  hello.c -E

This will print the preprocessed file to console


>cprime -config config.h hello.c

This will compile hello.c using the configuration file config.h and it
will ouput at hello.c

You can add your include paths at the config.h.
config.h works like it was included on the top of you input file.
